# AGENTS.md

## 项目概览

这是一个单页静态前端项目，名称为“游戏活动运营数据小助手”。项目目标是帮助游戏运营人员录入活动数据、计算关键运营指标、生成活动评级和复盘建议。

## 技术栈

- 使用原生 HTML、CSS、JavaScript。
- 不使用 npm、打包工具、框架或后端服务。
- 页面入口为 `index.html`。
- 样式集中在 `style.css`。
- 交互、计算、模板、导入解析逻辑集中在 `script.js`。

## 运行方式

- 直接双击 `index.html` 可运行。
- 如需本地服务，使用：

```powershell
python -m http.server 8000
```

## 代码约定

- 保持零构建依赖，优先使用浏览器原生能力。
- 不引入第三方库，除非明确需要并同步更新 README。
- 保持中文业务文案，代码标识符可继续使用英文。
- 新增功能优先放在 `script.js` 中对应模块附近，避免把业务逻辑写进 HTML。
- 样式继续沿用深色控制台主题和现有 CSS 变量。
- 表单字段新增时，需要同步更新字段采集、校验、导入别名、填充逻辑、README 字段说明。

## 关键功能位置

- `index.html`：页面结构、表单字段、按钮和结果展示区域。
- `style.css`：主题变量、布局、表单、按钮、指标卡片、评级卡片、Toast、导入区和响应式样式。
- `script.js`：
  - DOM 引用与表单数据采集。
  - 数据校验与指标计算。
  - 指标状态、综合评级、复盘文案生成。
  - 热门游戏活动模板数据。
  - 文件导入、粘贴导入、CSV/Markdown/XLSX 解析。

## 验证清单

修改后至少验证以下路径：

- 页面能正常打开，无阻塞报错。
- “填入示例数据”能填充表单。
- “生成复盘”能展示指标看板、评级和复盘报告。
- “清空数据”能清空表单并隐藏结果。
- 热门游戏活动模板能载入数据。
- 粘贴导入字段-值格式和表头-数据格式可识别。
- 移动端宽度下表单和指标卡片不明显溢出。

## 已知注意点

- `script.js` 中存在两段 `fillSampleData` 定义，当前行为由后定义版本生效；如后续重构，建议合并为一个定义。
- `generateReviewHTML` 当前在“活动基本情况”和“数据表现总结”中复用了同一段 `dataSummary`，如追求报告质量，可拆成基础信息和数据摘要两段。
- `.xlsx` 解析依赖浏览器 `DecompressionStream`，老旧浏览器可能不支持。
- `game-activity-assistant.zip` 是打包文件，不是源码主入口；源码以 `index.html`、`style.css`、`script.js` 为准。
